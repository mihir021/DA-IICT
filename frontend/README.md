# Frontend

This folder contains the browser UI for GroceryPulse.

- `admin.html` is the overview page
- `admin-operations.html` is the operations workspace
- `admin-insights.html` is the customer and admin intelligence workspace
- `js/api.js` is the single source of truth for all fetch calls
- charts are rendered with plain SVG, DOM, and CSS transforms
- admin login is stored in browser local storage until logout
